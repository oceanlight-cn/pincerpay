# X/Twitter Channel Rules

## Tone
Punchy, opinionated, occasionally dry humor. Confident but not arrogant. Every tweet should make a developer stop scrolling.

## Constraints
- **Single tweets**: Maximum 280 characters. Count carefully.
- **Threads**: Maximum 5 tweets. Each tweet max 280 characters.
- **Hashtags**: Maximum 2 per tweet. Prefer none unless genuinely useful for discovery.
- **Emojis**: Sparingly. One at most. Never emoji-heavy.

## Hook Patterns That Work
- Lead with a surprising number: "Stripe charges $0.30 per transaction. For a $0.01 API call, that's a 3,100% fee."
- Counterintuitive claim: "API keys are the new fax machines."
- Relatable pain point: "Every agent framework has a 'payment' gap. Nobody knows how to fill it."
- "We shipped X" + code snippet

## Thread Structure
1. **Hook** — attention-grabbing first tweet (number, hot take, or pain point)
2. **Evidence** — data, comparison, or technical explanation
3. **Mechanism** — how it works (code snippet or step-by-step)
4. **Result** — what you get (demo link, real output)
5. **CTA** — try it, read docs, join Discord (one link max)

## Anti-Patterns (NEVER do these)
- No "GM" or "gm" posts
- No engagement bait: "What do you think?", "Like if you agree", "Drop a 🔥"
- No hashtag spam: #Web3 #AI #Crypto #Payments #DeFi
- No emoji-laden hype: 🚀🔥💪🙌
- No vague promises: "Big things coming soon 👀"
- No corporate announcements: "We're thrilled to announce..."
- No price/token speculation (there is no token)

## Voice Examples

Good tweet:
"Your agent just paid $0.001 for weather data. Stripe would have charged $0.30. Math is math."

Bad tweet:
"🚀 Exciting news! PincerPay is revolutionizing the future of AI payments! Check out our amazing new feature! #AI #Web3 #Payments 🔥"

Good thread opener:
"Card rails charge $0.30 + 2.9% per transaction. For micropayments, that's a death sentence. Here's what we're doing about it:"

Bad thread opener:
"1/ We're so excited to share what we've been building at PincerPay! It's been an incredible journey and we can't wait to show you all the amazing things we've been working on 🧵👇"
