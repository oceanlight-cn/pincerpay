# Tweet Template

Output a single tweet. Maximum 280 characters. Count carefully.

Format:
```
[Tweet text — max 280 characters]
```

Rules:
- No hashtags unless genuinely useful for discovery (max 1)
- No emojis unless they add meaning (max 1)
- If including a link, assume it takes 23 characters (t.co wrapping)
- Line breaks are allowed and encouraged for rhythm
