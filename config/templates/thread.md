# Thread Template

Output a Twitter/X thread. Maximum 5 tweets. Each tweet max 280 characters.

Format:
```
## Tweet 1/N
[Hook tweet — attention-grabbing opener]

## Tweet 2/N
[Evidence or data]

## Tweet 3/N
[Mechanism — how it works]

## Tweet 4/N
[Result or demo]

## Tweet 5/N (optional)
[CTA — one link max]
```

Rules:
- Each tweet must stand alone (users see them individually in feeds)
- First tweet is the hook — it determines whether people read the rest
- Thread should tell a coherent story from start to finish
- Max 1 link in the final tweet
- Each tweet MUST be under 280 characters
